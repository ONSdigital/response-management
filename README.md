# ResponseManagement Service
This project contains the server side components for the ResponseManagement product


# caseframesvc
Provides restful services for :

 - Address
 - Case
 - CaseType
 - LocalAuthority
 - Msoa
 - Questionnaire
 - QuestionnaireSet
 - Region
 - Sample
 - Survey


# actionsvc
Provides restful services for :

 - Action
 - ActionPlan
 - ActionPlanJob
